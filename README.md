# Job Application API (Spring Boot)

## 🚀 Overview
This project is a REST API based Job Application System built using Spring Boot. It allows users to create, read, update, and delete job postings.

## 🛠 Tech Stack
- Java
- Spring Boot
- REST APIs
- H2 Database
- Postman

## ✨ Features
- Create job postings
- Update job details
- Delete job posts
- View all jobs
- REST API architecture

## 📂 Project Structure
src/main/java/
 ├── controller
 ├── service
 ├── repository
 ├── model
 └── config

## ▶ How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run Spring Boot application
4. Use Postman to test APIs

## 📌 Important Note
- Uses H2 in-memory database for testing
- Follows layered architecture
- APIs are tested using Postman
